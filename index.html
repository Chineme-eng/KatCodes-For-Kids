👅 LANGUAGES

What language should you learn first?

Computers speak lots of different languages! Here's how some popular ones say "Hello, World!" to the screen.

🌐 HTML — The skeleton. It gives websites their shape!

<!-- This is an HTML comment -->
<h1>Hello, World!</h1>
<p>I'm learning to code!</p>
<button>Click me</button>

🏁 Best for: Building your very first website. Start right here!

🎨 CSS — The paintbrush! It adds colors and cool designs.

/* Change the page background color */
body {
  background-color: coral;
  font-family: sans-serif;
  color: white;
}

h1 {
  font-size: 48px;
}

🏁 Best for: Making things look beautiful and stylish.

⚡ JavaScript — The brain! It makes buttons click and games work.

// Show a greeting
let name = "World";
console.log(`Hello, ${name}!`);

// React to a button click
document.querySelector("button")
  .addEventListener("click", () => {
    alert("You clicked me! 🎉");
  });

🏁 Best for: Web games, fun animations, and interactivity!

🐍 Python — Super easy to read. It looks almost like regular English!

# Print a greeting
print("Hello, World!")

# A simple function
def greet(name):
    return f"Hello, {name}! 👋"

print(greet("World"))

🏁 Best for: Cool science projects, math, and making robots think.

🍎 Swift — The language used to make iPad and iPhone games.

// Swift - used for iOS apps
import Foundation

let greeting: String = "Hello, World!"
print(greeting)

func greet(name: String) -> String {
    return "Hello, \(name)! 🍎"
}

🏁 Best for: Building your own apps to put on an iPhone or iPad.

🌙 HOW THIS PAGE WORKS

How the dark/light mode on this page works.

Want to add dark mode to your own site? Here's exactly how this page does it — it's simpler than you think!

1

CSS Variables for colors

Instead of writing color: orange everywhere, we use variables like --accent: #ff8c00 in :root {}. Then for dark mode, we just redefine those same variables.

2

A data attribute on <html>

The HTML element gets data-theme="light" or data-theme="dark". CSS uses [data-theme="dark"] { } to override the colors.

3

One tiny JavaScript function

A single function toggles the attribute: document.documentElement.dataset.theme = "dark". That's it! Smooth transitions are handled by CSS.

4

Remember the preference

We save the user's choice with localStorage.setItem("theme", "dark") so it persists on page reload.

/* Step 1: Define colors as variables */
:root {
  --bg: #fffaf3;
  --text: #2d2013;
  --accent: #ff8c00;
}

/* Step 2: Override for dark mode */
[data-theme="dark"] {
  --bg: #16131f;
  --text: #f0eaff;
  --accent: #f7c948;
}

/* Step 3: Use variables everywhere */
body { background: var(--bg); color: var(--text); }

/* Step 4: Smooth transitions */
body { transition: background 0.35s, color 0.35s; }
// JavaScript toggle function
function toggleTheme() {
  const html = document.documentElement;
  const isDark = html.dataset.theme === "dark";
  html.dataset.theme = isDark ? "light" : "dark";
  localStorage.setItem("theme", html.dataset.theme);
}

🐙 GITHUB PAGES

How to publish your website (with a parent!)

GitHub Pages lets you put your website on the real internet for free. Grab an adult and follow these steps:

1

Create a GitHub account

Go to github.com → click "Sign up". Pick a cool username, because it will be part of your web link!

2

Create a new repository

Click the + icon → "New repository" → name it my-cool-website → check "Add a README" → click "Create".

3

Upload your index.html

Click "Add file" → "Upload files" → drag your index.html file in → scroll down → click "Commit changes".

4

Enable GitHub Pages

Go to "Settings" → scroll to "Pages" on the left → under "Branch" select main → click "Save".

5

🎉 You are live!

Wait a minute or two, and your site will be at https://yourusername.github.io/my-cool-website. Show your friends!
